# Game Match

## Что это

Сервис подбора игр для компании друзей.

Пользователь создает комнату, отправляет ссылку друзьям, все проходят опрос, после чего сервис показывает топ игр для всей компании.

---

## Реализовано

* Создание комнаты
* Код комнаты из 6 символов
* Никнеймы участников
* Опрос из 8 вопросов
* Страница ожидания
* Результаты
* Steam карточки игр
* Алгоритм matchGames
* База generated-games.ts
* Локальный multiplayer flow через localStorage
