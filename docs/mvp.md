# Game Match MVP

## Product Vision

Game Match помогает компании друзей быстро выбрать игру, которая понравится большинству участников.

Основная проблема:

Когда друзья собираются играть вместе, они часто тратят больше времени на обсуждение игры, чем на саму игру.

Game Match сокращает время выбора за счет совместного опроса и автоматического подбора игр.

---

# User Flow

1. Пользователь создает комнату.
2. Получает уникальную ссылку.
3. Отправляет ссылку друзьям.
4. Игроки присоединяются к комнате.
5. Каждый проходит короткий опрос.
6. После завершения опроса всеми участниками запускается подбор.
7. Комната получает список наиболее подходящих игр.
8. Пользователи переходят на страницу игры в Steam.

# Matching Logic

Для MVP используется простой алгоритм.

Основа расчета:

* ответы игроков (Answer)
* теги игры (GameTag)
* допустимое количество игроков
* ограничения по бюджету

Алгоритм:

1. Собрать ответы всех участников комнаты.
2. Определить наиболее популярные предпочтения.
3. Сравнить предпочтения с тегами игр.
4. Рассчитать score для каждой игры.
5. Отсортировать игры по score.
6. Сохранить результаты в MatchResult.

# Success Criteria

MVP считается успешным если:

* Комнату можно создать менее чем за 10 секунд
* Игрок может пройти опрос менее чем за 1 минуту
* Компания из 2–6 человек получает подбор игр
* Пользователи понимают почему игра была рекомендована
* Пользователи могут перейти на страницу игры в Steam

# Not Included In MVP

Следующие функции не входят в первый релиз:

* Авторизация через Google
* Профиль пользователя
* UserGameVote
* UserPreference
* MatchHistory
* Tinder-механика
* Discord-интеграция
* Steam-синхронизация
* Система друзей
* Уведомления
* Платежи

Эти функции будут реализовываться после проверки основной гипотезы продукта.
