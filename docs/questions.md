# Questions Specification

## Purpose

Опрос используется для определения игровых предпочтений участников комнаты и последующего подбора игр.

Основная задача:

Собрать минимальное количество данных, достаточное для рекомендации игр компании друзей.

Опрос должен проходиться менее чем за 1 минуту.

# Matching Notes

При подборе игр необходимо учитывать:

* Совпадения по любимым жанрам
* Конфликты по нелюбимым жанрам
* Совпадения по игровым механикам
* Предпочтительный формат игры
* Сложность
* Бюджет
* Длину игровой сессии

Приоритеты для расчета score:

1. Disliked Genres
2. Format
3. Favorite Genres
4. Favorite Mechanics
5. Difficulty
6. Budget
7. Session Length

Игры, содержащие жанры, которые большинство игроков отметили как нежелательные, должны получать значительное снижение score.
